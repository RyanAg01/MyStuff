# Etiquetas\estructuras HTML

## Marcos de imágenes
Esta etiqueta se usa para mostrar una imagen con un marco al rededor
```html

<!-- '.etiqueta'  es el marco-->
<div class='etiqueta'> 
<!-- '.imagen'  es la imagen-->

    <img class='imagen'/>
</div>
```
